## Tentang Project

**Wuthering Waves Bahasa Indonesia** adalah mod komunitas yang bertujuan menerjemahkan konten cerita Wuthering Waves ke Bahasa Indonesia.

Project ini dibuat agar pemain Indonesia dapat menikmati story dan dialog game dengan lebih nyaman.

Repo ini merupakan bagian untuk melakukan lokalisasi utamanya dari en ke id. Repo ini tidak berdiri sendiri dan bergantung pada:

- [WuwaID](https://github.com/TitoTFP/WuwaID) (export lokalisasi menjadi file terstruktur)
- [WuwaIDLauncher](https://github.com/TitoTFP/WuwaIDLauncher) (launcher Wuthering Waves Indonesia)
